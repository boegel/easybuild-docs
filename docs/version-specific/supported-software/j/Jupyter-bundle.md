# Jupyter-bundle

This bundle collects a range of Jupyter interfaces (Lab, Notebook and nbclassic),   extensions (Jupyter Server Proxy, Jupyter Resource Monitor, Jupyter Lmod) and   the JupyterHub.

*homepage*: <https://jupyter.org/>
