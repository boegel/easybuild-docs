# bsddb3

bsddb3 is a nearly complete Python binding of the Oracle/Sleepycat C API for the Database Environment, Database, Cursor, Log Cursor, Sequence and Transaction objects.

*homepage*: <https://pypi.org/project/bsddb3/>
