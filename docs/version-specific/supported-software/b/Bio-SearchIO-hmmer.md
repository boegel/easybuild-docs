# Bio-SearchIO-hmmer

Code to parse output from hmmsearch, hmmscan, phmmer and nhmmer, compatible with both version 2 and version 3 of the HMMER package from http://hmmer.org.

*homepage*: <https://metacpan.org/pod/Bio::SearchIO::hmmer3>
