# Molden

Molden is a package for displaying Molecular Density from the  Ab Initio packages GAMESS-UK, GAMESS-US and GAUSSIAN and the Semi-Empirical  packages Mopac/Ampac

*homepage*: <http://www.cmbi.ru.nl/molden/>
